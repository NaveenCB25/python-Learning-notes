# Print Function in Python

## What is print()?

The `print()` function is used to display output on the screen.

It helps us show text, numbers, and variable values in the terminal.

---

# Basic Syntax

```python
print("Hello World")
```

---

# Hello World Program

```python
print("Hello World!")
```

## Output

```bash
Hello World!
```

---

# Strings in print()

Strings are text written inside:
- Single quotes `' '`
- Double quotes `" "`

## Example

```python
print('Python')
print("Programming")
```

---

# Printing Variables

```python
name = "Naveen"

print(name)
```

## Output

```bash
Naveen
```

---

# Printing Multiple Values

Use commas `,` to print multiple items.

## Example

```python
print("My favorite colors are", "blue", "green", "red")
```

## Output

```bash
My favorite colors are blue green red
```

Python automatically adds spaces between values.

---

# Example Program

```python
name = "Naveen"
age = 20

print("Name:", name)
print("Age:", age)
```

## Output

```bash
Name: Naveen
Age: 20
```

---

# Important Points

- `print()` displays output
- Text inside quotes is called a string
- Use commas to print multiple values
- Python adds spaces automatically
- `print()` is one of the most used functions in Python
