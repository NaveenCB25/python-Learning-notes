# Understanding Variables and Data Types

## 3. Declaring Variables

### What is Variable?
A variable stores data.

### Example

```python
name = "Naveen"
age = 20
```

---

### Variable Naming Rules

✅ Correct

```python
user_name = "Naveen"
_age = 20
```

❌ Wrong

```python
5name = "Error"
```

---

### Snake Case Naming

```python
user_age = 20
my_city_name = "Harihar"
```

---
